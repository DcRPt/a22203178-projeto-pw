Seletores CSS

CSS é usado para estilizar elementos HTML. Seletores determinam quais elementos são estilizados.

Seletor Usado Descrição
Elemento (tag) check_circle Seleciona elementos baseados no nome da tag, como <p> para parágrafos.
Classe check_circle Seleciona elementos com uma classe específica, indicado por um ponto (.), como .classname.
ID check_circle Seleciona um único elemento com um ID específico, indicado por uma hashtag (#), como #idname.
Atributo cancel Seleciona elementos com um atributo específico, como [type="text"] para campos de texto.
Pseudo-classe check_circle Seleciona elementos em um estado específico, como :hover quando o mouse está sobre o elemento.
Pseudo-elemento check_circle Seleciona partes específicas de um elemento, como ::first-letter para a primeira letra de um texto.

Combinadores de seletores

Combinador Usado Descrição
Descendente (espaço) check_circle Seleciona todos os elementos que são descendentes de um elemento específico, como div p para todos os <p> dentro de <div>.
Filho direto (>) check_circle Seleciona elementos que são filhos diretos de outro elemento, como ul > li para itens de lista diretamente dentro de uma lista.
Irmãos adjacentes (+) cancel Seleciona um elemento que é imediatamente seguido por outro, como h1 + p para um parágrafo imediatamente após um cabeçalho.
Irmãos gerais (~) check_circle Seleciona todos os elementos que são irmãos de um elemento, mesmo não sendo adjacentes, como h1 ~ p para todos os parágrafos que são irmãos de um <h1>.
